# Knowledge Commons: Local Governance

**Domain:** Local Governance  
**Network:** Regen Coordination  
**Maintained by:** Hub (aggregated from nodes)

---

## What This Domain Covers

- Local node governance structures and decision models
- Coordination patterns for bioregional actors
- Meeting and deliberation practices
- Member onboarding and role management
- Case studies of effective local governance in regenerative contexts

---

## Contributing

Nodes contribute knowledge to:
```
knowledge/local-governance/from-nodes/<node-id>/
```

---

## Key Questions This Domain Addresses

- How do local nodes coordinate without centralized control?
- What governance structures work for multi-stakeholder regenerative initiatives?
- How are roles and responsibilities managed across distributed teams?
- What decision models work for funding allocation at domain level?
