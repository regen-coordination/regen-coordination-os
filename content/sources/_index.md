## How to Use This Directory

1. **Finding Content**: Use the "Best For" column to identify which source is most relevant to your current work
2. **Cross-Referencing**: When writing or updating toolkit content, cite relevant sources using their code (e.g., "See Source A for implementation details")
3. **Contribution**: If you find additional sources that should be included, open a PR following the template in this directory

## Source Types

- **Toolkit**: Comprehensive resource with multiple content types
- **Guide**: Focused educational or instructional content
- **Course**: Structured learning material
- **Repository**: Code, templates, or technical resources
- **Book**: Long-form published content
- **Playbook**: Step-by-step implementation guides
- **Knowledge Base**: Wiki-style reference material
