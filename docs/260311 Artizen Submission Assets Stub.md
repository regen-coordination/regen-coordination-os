## Submission package (minimum)
1. Profile narrative (short)
2. Profile narrative (long)
3. Artifact framing + CTA
4. Evidence links table
5. Impact/accountability framing (auditable claims only)

## Skeleton
### Short narrative (100–150 words)
- RC identity
- Current cycle focus
- Why this matters now

### Long narrative (300–600 words)
- Network context
- Execution model
- What this cycle funds
- Accountability approach

### Artifact block
- Artifact title
- Problem statement
- Deliverable and acceptance criteria
- Call-to-action

### Evidence links
- Activities report
- Bread docs
- Website proof pages
- Supporting council notes (if needed)

## Risks to control
- Over-broad scope language
- Claims without linked proof
- Unclear cycle-specific deliverable
