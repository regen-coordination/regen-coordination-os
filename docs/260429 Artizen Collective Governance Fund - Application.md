# Artizen Collective Governance Fund — Application Draft (v3)

> **Note for the group:** Luiz only volunteered to post (non-US citizen requirement). This draft brings forward a primary anchor + modular add-ons + open questions, so the group can pick the scope. **Project lead and ongoing operator are TBD.**

---

## TL;DR

We propose to fund a **hybrid Regen Coordination network ideation event** — a follow-up to our December 2025 end-of-year retrospective — that convenes participants from ReFi DAO, GreenPill Network, and Bloom Network to **define concretely what Regen Coordination should be doing**, produce a slate of network proposals, and adopt **regen-coordination-os** (an org-os instance) as the shared collaborative environment for the work that follows.

The event executes at least one real collective decision (adopted focus areas + ratified proposals). Outputs are documented as a publicly shareable Cycle Report, usable by other networks.

## 1. Lead anchor — Regen Coordination Network Ideation Event

### What
A hybrid network convening that follows up on the **2025-12-12 end-of-year reflection workshop** (Slido + word cloud + forum thread, structured input format). The new event uses the same proven format and pushes one step further: from "reflection on the year past" to **"concrete proposals for what RC does next."**

The event is explicitly designed around an honest question the council has been circling for months (cf. 2026-04-03, 2026-04-24): *what is RC actually for now?*

### Format
- **One synchronous hybrid session** (~2 hours, online via Google Meet + co-located node clusters in cities where members are convening anyway — e.g. Barcelona, NYC).
- **Pre-event prep:** open Notion + forum working space; Matt's in-flight budget/focus synthesis (per 2026-04-24 council) as one input among several.
- **Live tools:** Slido for structured polling and word-cloud generation (precedent from 2025-12-12); Notion / forum thread for written contributions; Telegram for async tail.
- **Output:** a slate of proposals (focus areas + concrete initiatives) to route to the network for review, and where applicable, conviction-voting on existing Gardens pools.

### Decision executed
- Council formally adopts H2 2026 focus areas, with named owners.
- At least one concrete proposal ratified for execution (e.g. regen-coordination-os adoption as the shared collab environment; or a specific Gardens-pool allocation).
- Decisions logged on-chain where applicable; otherwise in `MEMORY.md` and the public meeting record.

### Why this fits the Fund
The Collective Governance Fund seeks **community-led, hybrid, real-decision-making projects using modern governance tools, with documented outcomes.** This event is exactly that — and is grounded in a precedent we already executed (the December 2025 retrospective).

### What the funding actually pays for
- Facilitation + design time for the hybrid format.
- Slido / Notion / forum infrastructure for the cycle.
- Travel/co-location stipends for at least 2 node-clusters (e.g. Barcelona, NYC, BR/MX).
- Documentation work: Cycle Report producer + on-chain artifact issuance.
- Handover materials: regen-coordination-os onboarding kit so the work continues after the event.

Active council stewards (recurring participation across multiple meetings in the last 12 months) span ReFi DAO, GreenPill Network, Bloom-aligned, and partner orgs — well above 7. The 2025-12-12 retrospective itself drew a larger network audience via Slido + forum (precedent for the format scaling).

## 4. Why this fund, why now

**Why this fund:** The Collective Governance Fund's purpose is governance events + participatory budgeting tools. Our proposed cycle is exactly that — community-led, hybrid, modern-tooled, real-decision, documented. We do not need Artizen to fund an idea; we need it to underwrite a single well-instrumented cycle of work we already do, so the methodology + documentation are clean enough to reuse.

**Why now:** The council has spent the last six weeks circling the question *what concretely should RC be doing* (2026-04-03, 2026-04-24). Matt's budget synthesis is in flight. The 2025-12-12 retrospective format is proven. Doing a focused follow-up convening this quarter is the right move regardless of funding — but funding turns it from "another council call" into a real instrumented event with a public deliverable.

**Disclosure:** Matt is the fund manager for the Collective Governance Fund and is also a council steward. We mention this transparently; the application is judged on its merits.
