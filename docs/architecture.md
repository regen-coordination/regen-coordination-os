# Coop Architecture

## Layers

1. Local layer: extension and PWA keep local-first artifacts.
2. Shared membrane: peer replication and consensus through anchor-assisted signaling.
3. Cold storage: approved artifacts archived through Storacha/Filecoin.

## Compute model

- Standard nodes: capture, summarize, and sync.
- Anchor nodes: stronger inference, key management, and integrations.

## On-chain layer

- Coop registry contract for membership and discoverability.
- Smart account session keys for constrained agent actions.
