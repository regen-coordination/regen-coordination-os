# Coop Grant Landscape — March 2026

We've mapped out the grants ecosystem across web3, AI, privacy, and open-source funding. Here's where Coop fits and what's actionable now.

## URGENT: Apply Within 10 Days

### 1. NLnet NGI Zero Commons Fund — April 1, 2026
- **Size:** EUR 5K–50K (scalable)
- **Apply:** https://nlnet.nl/propose/
- **Why #1:** NLnet's March 2026 batch just funded 44 projects including Yrs (Yjs persistent documents) and NextGraph (local-first P2P with CRDTs). Coop is the next project in that lineage. They've funded browser extensions before (WorldBrain.io, NoScript). Lightweight short-form application.
- **Angle:** Local-first CRDT coordination tool with privacy-preserving group identity. Browser extension as sovereignty-preserving knowledge commons.

### 2. NLnet NGI Zero Core — April 1, 2026
- **Size:** EUR 5K–50K
- **Apply:** Same portal — https://nlnet.nl/propose/
- **Angle:** Frame Coop's CRDT sync protocol, y-webrtc transport, and passkey auth as core internet infrastructure improvements. Different emphasis than Commons Fund — apply to both.

## HIGH PRIORITY: Rolling / Near-Term

### 3. Ethereum Foundation ESP — Wishlist: Smart Account Passkey Support
- **Size:** $50K–200K (estimated)
- **Apply:** https://esp.ethereum.foundation/applicants/wishlist/passkey-support
- **Why:** This wishlist item literally describes what Coop already builds — "passkey authentication in ERC-4337 smart accounts without centralized wallet infrastructure." Direct hit.
- **Status:** Rolling, open now.

### 4. ESP RFP: Economic & Organizational Models
- **Apply:** https://esp.ethereum.foundation/applicants/rfp
- **What they want:** "Novel economic and organizational models at the application layer made possible by decentralized, credibly neutral trust infrastructure."
- **Angle:** Cooperative governance with Safe multisig, ZK anonymous publishing, P2P knowledge coordination.

### 5. Safe Ecosystem Foundation
- **Size:** From $100M pool (~EUR 25K average in Wave 1)
- **Contact:** Direct inquiry via https://forum.safe.global
- **Angle:** Consumer Safe adoption showcase. Passkey-first identity → Safe smart account owners. Reusable passkey-to-Safe tooling.
- **Status:** No announced 2026 wave yet — reach out proactively.

### 6. Filecoin Open Grants
- **Size:** Up to $50K
- **Apply:** GitHub issue at https://github.com/filecoin-project/devgrants
- **Angle:** Real consumer application driving Filecoin storage demand through community knowledge archival via Storacha.
- **Status:** Rolling, 3-month review cycles.

### 7. Arbitrum Foundation dApps Grant
- **Size:** $20K–150K (milestone-based)
- **Apply:** https://arbitrum.foundation/grants
- **Note:** Requires testnet/mainnet deployment.
- **Status:** Rolling.
