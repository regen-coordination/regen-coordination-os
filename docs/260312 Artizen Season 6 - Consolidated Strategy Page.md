## Unified Strategy

## 1) Application architecture
Use **one base application narrative**, then adapt per-fund overlays.

**Base narrative:**
- RC coordinates regen ecosystem actors and funding flows.
- Toolkit is the practical delivery layer for mechanism design and implementation support.
- Work includes reusable knowledge resources, onboarding pathways, and implementation templates.

**Per-fund overlays:**
- Reuse same base copy.
- Customize only: “Why this project is eligible for this fund.”
- Apply broadly across relevant regen and AI-adjacent funds.

## 2) Funding split principle
- Initial working anchor discussed: **50% of RC funding toward this project** (flexible).
- Final % can be adapted based on inflows and council alignment.

## 3) Positioning
Adopt a **hyper-specific niche** positioning:
- Focus: toolkit and mechanism-design support for this community.
- Avoid duplicating broader bioregional knowledge-commons scope being developed elsewhere.
- Integrate externally where useful; specialize locally where differentiation is strongest.
