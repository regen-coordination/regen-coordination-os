## Observations

- **agentskills.io is NOT a skill marketplace** — it's a format-specification hub. Should be removed from `skills-curator` skill's default sources (action item for Phase 6 retrospective).
- **Gaps surfaced**: no skills found for shadcn-installer, astro-bootstrap, dedicated a11y-audit, Zod-skill, data-adapter skills. Each has a manual mitigation. All are candidates for new skills to author and promote in Phase 6.
- **`npx skills add -a '*'`** creates ~36 harness-symlink directories at the repo root (`.aider-desk/`, `.augment/`, etc.) — gitignored as regenerable via `npx skills experimental_sync`.
- **Brand reality check**: the sky-to-sun gradient is fundamentally incompatible with the original MASTERPLAN's dark-slate ReFi-DAO-style palette. The aggregator design pivots accordingly — DESIGN.md (Phase 1) will be light-default with dark as derived variant.
