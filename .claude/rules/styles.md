# Style Rules

- CSS uses plain class-based styling with brand color custom properties.
- Brand palette:
  - `--coop-cream: #fcf5ef` — Default background
  - `--coop-brown: #4f2e1f` — Primary text
  - `--coop-brown-soft: #6b4a36` — Secondary text
  - `--coop-green: #5a7d10` — Active states, knowledge/growth
  - `--coop-orange: #fd8a01` — CTAs, publish/archive emphasis
  - `--coop-mist: #d8d4d0` — Neutral backdrop
- Visual: warm (not corporate), observant (not surveillance-like), structured (not overdesigned).
- Key class conventions: `nest-card`, `lens-card`, `state-pill`, `board-chip`, `coop-board-node`.
- Extension UI: dense but warm (cream/mist base, brown text, green for routing, orange for publish).
- Landing page: more atmospheric (nest cards, lens cards, timeline steps, glow mark).
