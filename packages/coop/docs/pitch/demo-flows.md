# Demo Flows

## Demo 1: Local Coop (ReFi Barcelona style)

1. Create Coop and invite one member with share code.
2. Capture city grant resources from open tabs.
3. Dictate local cleanup needs via voice.
4. Run impact-reporting skill and review output.
5. Archive approved report to Filecoin reference.

## Demo 2: Online Coop (GreenPill region style)

1. Join Coop with existing share code.
2. Share curation resources from multiple tabs.
3. Run coordination skill for action extraction.
4. Draft governance proposal from synthesized notes.
