# Coop Onboarding Flow

## Step 1: Discovery call

- Capture organization context and local needs.
- Map needs to four pillars.

## Step 2: Create Coop

- Generate Coop share code.
- Register initial members.
- Configure anchor node endpoint.

## Step 3: Ingest first context set

- Upload/capture meeting notes.
- Dump resource tabs.
- Add first voice dictation.

## Step 4: Activate first skill

- Run impact-reporting or coordination skill.
- Review output and approve for archive.
