# Pricing

Coop does not have a public pricing table yet.

## What Exists Today

The repo and docs describe a product that is still in active buildout. In practical terms, access
today looks like one of these paths:

- local development from the open repository
- trusted-tester extension installs outside the Chrome Web Store flow
- guided demo and deployment setups while the product surface is still stabilizing

## What This Means For A Community

If you are evaluating Coop right now, the honest expectation is:

- there is not a self-serve commercial pricing page to compare
- some flows are oriented around builders, testers, and early operators
- live archive and onchain paths depend on environment setup and credentials

## What We Are Not Claiming

This page intentionally does not invent:

- paid tiers
- seat pricing
- enterprise plans
- public SLAs that are not actually documented elsewhere in the repo

## The Practical Read

Today, Coop is best understood as an actively developed coordination product with an open codebase,
an early-access installation posture, and a growing public docs surface.

If that is the stage you want, the best next pages are [How It Works](/how-it-works),
[Creating A Coop](/creating-a-coop), and [Builder Getting Started](/builder/getting-started).
